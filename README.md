# 💼 Pure HTML Portfolio Website

A simple personal portfolio website built entirely with pure HTML. This project showcases my profile, educational background, extracurricular activities, projects, and social links using only HTML elements without any external CSS frameworks or JavaScript.

🔗 **Live Demo:** [View Portfolio](https://pure-html-portfolio.vercel.app/)  
🐙 **GitHub Repo:** [View Repository](https://github.com/dsdipu/pureHtmlPortfolio)

---

## 📌 Overview

This portfolio website was developed as a practice project to strengthen fundamental HTML concepts and webpage structuring. The website consists of multiple pages connected through navigation links and presents personal information in a clean and organized layout.

### Key Highlights:
- ✅ Built using pure HTML only
- ✅ Multi-page website structure
- ✅ Personal portfolio presentation
- ✅ Educational qualification showcase
- ✅ Project listing page
- ✅ Social media and contact information
- ✅ Image integration using local assets
